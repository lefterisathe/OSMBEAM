## What OSMBeam Does

- Lets you select a real-world area or route.
- Fetches OpenStreetMap data for roads and map features.
- Processes terrain and elevation data.
- Converts the data into BeamNG-compatible map files.
- Packages the final result as a mod you can install in BeamNG.drive.
